Fixpath

A handy script to fix PATH issues, like repeated or unwanted entries.

Download .zip Download .tar.gz View on GitHub

Motivation.

fixpath is a set of scripts to fix the order and remove repeated and unwanted entries in the PATH or MANPATH. It was originally intended to deal with issues in X11 terminals on Mac OS 10.8 for Fink, but could easily be adapted for other scenarios.

To illustrate this, here are what the PATH environment variable shows up as on my Mountain Lion system:

OS X "native" terminal

$ printenv PATH
/sw/lib/perl5/ExtUtils:/sw/bin:/sw/sbin:
/usr/bin:/bin:/usr/sbin:/sbin:/usr/local/bin:
/opt/X11/bin:/Users/hansen/bin

X11 terminal

$ printenv PATH
/usr/bin:/bin:/usr/sbin:/sbin:/usr/local/bin:/opt/X11/bin:
/sw/lib/perl5/ExtUtils:/sw/bin:/sw/sbin:/Users/hansen/bin

For the sake of consistency, I'd prefer to have the same PATH regardless of what terminal I'm using, and so I created the fixpath tool.

Installation:

In Fink

Use fink install fixpath

Outside of Fink

Currently there isn't an installer. To set things up for your use, replace the string @PREFIX@ in fixpath.pl.in, fixpath.sh.in and fixpath.csh.in by a directory that you would like to have take precedence in the PATH or MANPATH. For example:

#!/bin/sh
for ext in pl csh sh ; do
    sed -e 's,@PREFIX@,/sw,g' fixpath.$ext.in > fixpath.$ext
done
